# INF01126 - Estruturas De Dados I (Turma U 2025/2)

Do enunciado:
> Desenvolver um jogo de caçada no labirinto, ao estilo Pac-Man, no qual o jogador se movimenta por um mapa bidimensional, coleta itens e enfrenta criaturas que o perseguem utilizando algoritmos de busca. O foco do trabalho é aplicar conceitos fundamentais de estruturas de dados, como TADs, listas, pilhas, filas e algoritmos de busca, além de modularização, documentação e legibilidade de código.
